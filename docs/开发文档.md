## 二、项目概述

### 2.1 背景
面向四川省 67 个民族县 3000 名基层卫生技术人员（乡镇卫生院、社区卫生服务中心七类卫技人员：临床医学、公共卫生、护理、医技 B 超/心电/检验/影像），提供 MOOC 学习、在线考试、离线学习、实时咨询等网络培训服务，每人每年 200 学时。

### 2.2 项目性质
课程期末作业项目。**不需要**生产级高并发/安全合规，但 docx 第 4 节的高并发技术方案（CDN/LVS/读写分离/分库分表/MapReduce/流媒体分离）需做**代码级概念验证**，云服务三层架构（IAAS/PAAS/SAAS）需做**文档分层设计**。

### 2.3 三种角色

| 角色 | 权限 | 默认账号 |
|------|------|---------|
| **管理员** | 全部功能：用户/课程/考试/计划/统计/咨询管理 | admin / 123456 |
| **讲师** | 课程管理、试题管理、查看学员进度、在线答疑 | teacher01 / 123456 |
| **学员** | 课程学习、在线考试、查看学习记录、咨询问答 | student01 / 123456 |

### 2.4 技术栈

| 模块 | 技术 | 版本 | 说明 |
|------|------|------|------|
| **后端** | Spring Boot | 2.7.18 | 单体应用（非微服务） |
| **ORM** | MyBatis-Plus | 3.5.3 | 数据访问 |
| **数据库** | MySQL | 8.0 | 单机 + 读写分离概念验证 |
| **缓存** | Redis | 7.x | 模拟 CDN 热点缓存 |
| **认证** | JWT | jjwt 0.11.5 | Token 认证 |
| **前端** | Vue3 + Element Plus | 3.x | 后台管理系统 |
| **小程序** | 微信小程序原生 | — | 学员端 |
| **构建** | Maven | 3.6+ | 多模块构建 |
| **包管理** | npm | 随 Node 18 | 前端依赖 |

## 五、公共层速览（training-common）

公共层代码完整实现见 `dev-backend.md` 第 1 章。这里给出关键类职责速查：

| 类/包 | 路径 | 职责 |
|-------|------|------|
| `Result<T>` | common/result | 统一响应包装 `{code, message, data}` |
| `ResultCode` | common/result | 响应码枚举（200/401/403/1001/1002/1005/1007） |
| `BusinessException` | common/exception | 业务异常（带错误码） |
| `GlobalExceptionHandler` | common/exception | 全局异常处理（@RestControllerAdvice） |
| `JwtUtils` | common/utils | JWT 生成/解析/校验（HMAC-SHA256） |
| `entity/` | common/entity | 实体类（SysUser/Course/Exam 等 17 个） |
| `dto/` | common/dto | 数据传输对象 |
| `vo/` | common/vo | 视图对象（返回前端） |

### 统一响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": { ... }
}
```

分页 data 为 `{ records: [], total: N }`。

### 错误码

| 码 | 含义 |
|----|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未登录或 Token 过期 |
| 403 | 无权限 |
| 500 | 服务端错误 |
| 1001 | 用户名已存在 |
| 1002 | 用户名或密码错误 |
| 1005 | 已报名该课程 |
| 1007 | 已达到最大重考次数 |

## 六、数据库速览

完整 17 张表结构 + DDL + 约束见 `dev-database.md`。核心表：

| 表名 | 说明 | 关键关联 |
|------|------|---------|
| `sys_user` | 用户表 | 三角色（admin/teacher/student） |
| `course` | 课程表 | **offline_flag** 离线属性标记（v2.0 新增） |
| `exam` | 考试表 | **exam_type** 区分 3 种考试类型 |
| `question` | 试题表 | **question_type** 5 种题型 + **difficulty** 3 种难度 |
| `study_record` | 学习记录 | 唯一键（student+course+chapter）+ 断点续播 |
| `consult_record` | 咨询记录 | **transfer_time/reply_time/sla_exceeded** SLA 计时（v2.0 新增） |
| `knowledge_base` | 知识库 | 关键词匹配智能问答 |

### v2.0 新增字段（需在 database.sql 基础上补充）

```sql
ALTER TABLE course ADD COLUMN offline_flag TINYINT DEFAULT 0 COMMENT '是否可离线：0否 1是';
ALTER TABLE course ADD COLUMN zip_url VARCHAR(255) DEFAULT NULL COMMENT '离线ZIP下载路径';
ALTER TABLE consult_record ADD COLUMN transfer_time DATETIME COMMENT '转人工时间（SLA起点）';
ALTER TABLE consult_record ADD COLUMN reply_time DATETIME COMMENT '人工回复时间';
ALTER TABLE consult_record ADD COLUMN sla_exceeded TINYINT DEFAULT 0 COMMENT 'SLA超时：0否 1是';
```
