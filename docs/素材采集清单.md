## 一、盘点结论（先看这段）

**好消息：项目已做好「图片缺失不出错」的准备。**
代码里凡涉及用户数据图片（课程封面、头像）都用 `v-if` / `el-empty` / 默认值兜底——所以前端**一张图都不放也能跑通所有功能**。

**目前引用真实图片的只有 2 处，都是后台管理端自带的静态图：**

| 文件 | 图 | 尺寸建议 | 来源 |
|---|---|---|---|
| `training-admin/frontend/src/views/dashboard/index.vue` dashboard 欢迎横幅背景 | 横幅/插画 | 1200×200，浅蓝+医疗元素 | 代码叫 `welcome-banner`（已是 CSS 蓝渐变，没图也 OK） |
| `web-student/src/views/home/index.vue` 首页推荐课程卡 `.rec-card__cover` | 课程封面兜底图 | 320×180 | 代码已经用 `v-if="course.coverUrl"` —— 没图会显示 el-empty 占位，页面不崩 |

**真正需要你动手加图的是小程序**：主站 `pages/index/index.js` 有 **11 处硬编码本地图片**：

| 引用位置 | 文件名 | 建议尺寸 | 备注 |
|---|---|---|---|
| 顶部轮播 | `images/banner1.jpg` | 750×300，2~3 张 | 必填，不然轮播空白 |
| 顶部轮播 | `images/banner2.jpg` | 750×300 | 同上 |
| 首页金刚区图标 | `images/icon-course.png` | 64×64 PNG | 课程入口 |
| 首页金刚区图标 | `images/icon-exam.png` | 64×64 PNG | 考试入口 |
| 首页金刚区图标 | `images/icon-plan.png` | 64×64 PNG | 计划入口 |
| 首页金刚区图标 | `images/icon-my.png` | 64×64 PNG | 个人中心 |
| 首页金刚区图标 | `images/icon-consult.png` | 64×64 PNG | 咨询入口 |

**结论**：小程序是图片大户，学员网页几乎不需要加图（走的是后端返回的 `coverUrl`）。

## 二、给后端补一点「封面 URL」数据（必做）

后台 `/admin/course/page` 返回的 `coverUrl` 现在是空字符串 → 学员网页的课程卡会显示 element-plus 默认空占位。补一条 SQL：

```sql
UPDATE course SET coverUrl='https://picsum.photos/seed/course1/400/225' WHERE id=1;
UPDATE course SET coverUrl='https://picsum.photos/seed/course2/400/225' WHERE id=2;
UPDATE course SET coverUrl='https://picsum.photos/seed/course3/400/225' WHERE id=3;
UPDATE course SET coverUrl='https://picsum.photos/seed/course4/400/225' WHERE id=4;
```

> `picsum.photos` 是免费占位图服务，答辩时不作商用。`course1` 种子固定就会一直返回**同一张**图，不会每次刷新都变，截图素材稳定。

换真实项目图时把 URL 改成你自己的 OSS/COS 链接即可。

## 三、补小程序轮播图（7 张，做完就有完整首页）

在 `miniprogram/images/` 下放 7 个文件：

```bash
miniprogram/
└── images/
    ├── banner1.jpg       # 750×300  培训平台主视觉
    ├── banner2.jpg       # 750×300  民族地区基层卫生宣传
    ├── banner3.jpg       # 750×300  继续教育/培训主题（可选，轮播第 3 帧）
    ├── icon-course.png   # 64×64   课程
    ├── icon-exam.png     # 64×64   考试
    ├── icon-plan.png     # 64×64   计划
    ├── icon-my.png       # 64×64   我的
    └── icon-consult.png  # 64×64   咨询
```

### 怎么获取

**方案 A：用 AI 一键生成（推荐，1 分钟搞定）**

| 类型 | 提示词（直接粘给豆包/通义/Claude） | 张数 |
|---|---|---|
| 横幅 | `画一张横版插画：四川藏族羌族自治州乡村卫生院场景，浅蓝色调，培训学习氛围，750×300 无文字无 logo` | 3 |
| 图标 | `扁平化图标 design，simple flat icon，line style，#1677ff 蓝色，白色背景：听诊器/书本/试卷/表格/用户/对话气泡` | 5 张 64×64 |

**方案 B：从免费素材站下载**

- 横幅：<https://unsplash.com/s/photos/medical> → 选横版图，裁剪 750×300
- 图标：<https://www.iconfont.cn/search?word=课程&type=illustration>（支持 PNG 导出）

**方案 C：用 picsum 占位（最快但答辩时建议换成真实图）**

把 `pages/index/index.js` 第 6-7 行改成：
```js
{ id: 1, image: 'https://picsum.photos/seed/banner1/750/300' },
{ id: 2, image: 'https://picsum.photos/seed/banner2/750/300' }
```
即可跳过图片文件。
